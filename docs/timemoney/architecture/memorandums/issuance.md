# Меморандум эмиссии

Меморандум эмиссии включает собой 
подписанный всеми тремя серверами
`bok` ключ и 
список вида:
1. bank_id
2. banknote_id
3. code
4. amount

Так же меморандум включает в себя служебные поля:
* все `mok` ключи
* время меморандума
* произвольный комментарий `comment`
